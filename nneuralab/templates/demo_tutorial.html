Verification Demo Tutorial

With the verification demo you have the opportunity to test and play around with our watermarking package. It serves as a playground for anyone interested in the watermarking of neural networks. In this tutorial you'll learn how to use this demo and have fun with watermarking yourselfs! 😎


There are 3 easy steps you need to take for working with our verification demo:

  1. Watermark your deep learning model with the neuralwatermark package and put the returned ownership information into a pickel file
  2. Build a prediction API for your model
  3. Enter the API-link and yourn ownership file to the verification demo and you're ready to go!

1. Watermark your DL model

In this tutorial we skip the part of how to watermark your DL model since all the information is already contained in the neuralwatermark documentation. So feel free to visit!

You also need to store your watermarked model in order to use it for the prediction API. For example you could store it in a pickel file with the following command:


torch.save(model.state_dict(), PATH)

Now that your model is saved we can go on to build a prediction API for your model!



2. Building prediction API

Now that you have the model ready, you need to build a prediction API for your model. For the verification demo you need to build this API with the Django REST framework. For our core service however, it doesn't matter how the API of your suspect was built. This tutorial will mainly focus on building a prediction API with the Django REST framework

This requires the installation of django as well as the django rest framework. Just execute the below commands in your terminal and you're ready to go!


pip install django

pip install django-rest-framework


Now that your have everything installed you need to setup a django project and create a django app. For this we recommend taking a look at this tutorial on the official django webpage

3. Enter information and get started!

Just go to the verification demo, enter your ownership information and the API link and get started!
